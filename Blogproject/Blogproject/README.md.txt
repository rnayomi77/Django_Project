Django User Authentication Project
This Django project is a simple web application that enables user registration, login, logout, and provides a home page. 
It utilizes the Django authentication system for managing user-related functionalities.


## User Registration:

Users can register with a unique username and password.
Handles cases where the chosen username is already taken.

## User Login:
Registered users can log in with their credentials.
Utilizes Django's built-in authentication forms for secure login.

## User Logout:

Allows authenticated users to log out and redirect to the home page.
Home Page:

Provides a simple home page for users to access once logged in.
